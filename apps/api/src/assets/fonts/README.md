# פונטים עבריים ל-PDF

כדי להוסיף תמיכה בעברית ב-PDF, יש להוריד את הפונט Noto Sans Hebrew.

## הוראות:

1. הורד את הפונט Noto Sans Hebrew מ-Google Fonts:
   https://fonts.google.com/noto/specimen/Noto+Sans+Hebrew

2. הורד את הגרסה Regular (400) והגרסה Bold (700)

3. שמור את הקבצים הבאים בתיקייה הזו:
   - `NotoSansHebrew-Regular.ttf`
   - `NotoSansHebrew-Bold.ttf`

או הורד ישירות מ-:
https://github.com/google/fonts/raw/main/ofl/notosanshebrew/NotoSansHebrew%5Bwght%5D.ttf

## חלופה:

אם לא רוצה להשתמש ב-Noto Sans Hebrew, אפשר להשתמש בפונט עברי אחר כמו:
- Arial Hebrew (אם מותקן במערכת)
- David (אם מותקן במערכת)
